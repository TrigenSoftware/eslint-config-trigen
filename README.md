Trigen's ESLint config.

## Install

```bash
npm i -D eslint-config-trigen
# or
yarn add -D eslint-config-trigen
```

## Configure

Add `'extends': 'trigen'` (to lint code with ReactJS) or `'extends': 'trigen/base'` to your `.eslintrc`.
